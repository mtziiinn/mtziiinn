# Olá, eu sou o Matheus Gonçalves! 👋

Sou de São Paulo - SP e estou cursando o 2º semestre de **Análise e Desenvolvimento de Sistemas (ADS)**. Focado em desenvolvimento **Full Stack** com ênfase em Backend — construo desde APIs seguras e escaláveis até plataformas SaaS completas.

---

### 💻 Sobre Mim

- 🏗️ Construo sistemas completos: CRMs, plataformas SaaS, APIs REST com autenticação robusta e bots para Discord.
- 🔐 Preocupado com segurança desde o início: JWT com refresh tokens, rate limiting com Redis, RLS no Supabase, validação server-side.
- 🧪 Escrevo testes automatizados com Jest + Supertest (~85% de cobertura na Pro-Auth-API).
- 🌱 2º semestre de ADS — aprendendo rápido e construindo projetos reais.
